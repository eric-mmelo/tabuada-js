# tabuada-js
📌 Project Description – JavaScript Multiplication Table Generator

This project is a simple and interactive web application that generates multiplication tables. Built using HTML, CSS, and JavaScript, it allows users to enter a number and define how far they want the multiplication to go, dynamically displaying the results on the screen.

🚀 Features

-Input field to enter a number
-Customizable multiplication limit (default set to 10)
-Dynamic table generation without page reload
-Clean, responsive, and user-friendly interface
-Organized visual output displaying each multiplication operation

💻 Technologies Used

-HTML5 – Application structure
-CSS3 – Styling with a centered layout and gradient background
-Vanilla JavaScript – DOM manipulation and dynamic content rendering

🎯 Project Purpose

The main goal of this project is to practice:
-Handling form events (submit event)
-Capturing user input
-DOM manipulation
-Loop structures
-Basic UI structuring and styling with CSS
